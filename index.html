This is an example mashup using the new table

This mashup consists of a listbox and a table, all fetching data from the same app.

To run this example you need to be logged in to the appropriate tenant and allow third party cookies.

Instructions

Use tab to navigate between links and elements as a normal webpage.
When you hit the first cell in the table:
  • Use the arrow keys to move around.
  • Space to select a cell.
  • Enter to confirm selection.
  • Escape to cancel selections.
